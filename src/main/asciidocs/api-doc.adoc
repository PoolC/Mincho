`POST /member`, `POST /login` 을 제외한 모든 API는 토큰 발급 절차를 거쳐야 한다. +
모든 HTTP 상태 코드는 https://tools.ietf.org/html/rfc7231#section-6[RFC 7231] 문서를 따른다.

== 인증
